# Coffee Shop Management (Avalonia, .NET 9)

[![.NET CI](https://github.com/Keimhean/oodafinal-c-AvaloniaUI/actions/workflows/dotnet-ci.yml/badge.svg)](https://github.com/Keimhean/oodafinal-c-AvaloniaUI/actions/workflows/dotnet-ci.yml)

ប្រព័ន្ធគ្រប់គ្រងហាងកាហ្វេដែលសង់ដោយ .NET 9 និង Avalonia UI (cross-platform) — គ្រប់គ្រងផលិតផល ការបញ្ជាទិញ និងមានការតំឡើងដាតាបេសដោយស្វ័យប្រវត្តិ។

- មើលឯកសារព័ត៌មានលម្អិត (English + Khmer):
  - docs/uml/README_DETAILED.md

- គន្លឹះបច្ចេកទេស:
  - .NET 9, Avalonia UI 11, CommunityToolkit.Mvvm
  - SQL Server (Docker), Microsoft.Data.SqlClient
  - MVVM + Dependency Injection

- របៀបដំណើរការ (សង្ខេប):
  1) ចាប់ផ្តើម DB: `docker-compose up -d`
  2) run'app':
     - `cd CoffeeShopManagement`
     - `dotnet run`
